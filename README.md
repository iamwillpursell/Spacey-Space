# Spacey Space

*Introduction*\
Spacey Space is a simple app that uses the NASA APOD api to get an image, title, date, and description, from NASA to display on the website. The data is updated by NASA every 24hrs.

*Tools Used*
* NASA APOD API
* React Router
* Netlify
